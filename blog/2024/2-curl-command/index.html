pipe

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  • The pipe | takes the output of the curl command, which is the Rust installer script, and passes it to sh, the Unix shell, to execute the script.
  • In summary, the | sh means that the downloaded script is immediately executed in the shell.
  • This is a common way to install software from the internet, but you should always be cautious and ensure you trust the source, as running scripts directly can be a security risk.
